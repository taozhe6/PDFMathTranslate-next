    Installa PDFMathTranslate tramite docker

    Cos'è Docker?

    Docker è una piattaforma open per lo sviluppo, la distribuzione e l'esecuzione di applicazioni. Docker consente di separare le tue applicazioni dall'infrastruttura, permettendoti di fornire software rapidamente. Con Docker, puoi gestire la tua infrastruttura nello stesso modo in cui gestisci le tue applicazioni. Sfruttando le metodologie di Docker per la distribuzione, il test e l'implementazione del codice, puoi ridurre significativamente il ritardo tra la scrittura del codice e la sua esecuzione in produzione.

    Installazione

    1. Esegui il pull e avvia:

    docker pull awwaawwa/pdfmathtranslate-next
docker run -d -p 7860:7860 awwaawwa/pdfmathtranslate-next

    Note

    • Se non riesci ad accedere a Docker Hub, prova l'immagine su GitHub Container Registry.
    docker pull ghcr.io/PDFMathTranslate/PDFMathTranslate-next
docker run -d -p 7860:7860 ghcr.io/PDFMathTranslate/PDFMathTranslate-next

    2. Inserisci questo URL nel tuo browser predefinito per aprire la pagina WebUI:

    http://localhost:7860/

    Note

    Se riscontri problemi durante l'utilizzo dell'interfaccia Web, consulta Utilizzo --> WebUI.

    Note

    Se riscontri problemi durante l'utilizzo della riga di comando, consulta Utilizzo --> Riga di comando.
