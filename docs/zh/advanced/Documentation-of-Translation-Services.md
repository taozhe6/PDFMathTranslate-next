### 通过命令行查看可用的翻译服务

您可以通过在命令行中打印帮助信息来确认可用的翻译服务及其使用方法。

```bash
pdf2zh_next -h
```

在帮助信息的末尾，您可以查看不同翻译服务的详细信息。

### 翻译引擎支持政策

#### Tier 1 (官方支持)

**Tier 1 翻译引擎**由项目维护者直接维护。虽然维护者**不经常使用此项目**，但他们会依赖 GitHub issues 来发现问题。当这些引擎中的任何一个遇到问题时，维护者会尽快修复，以确保稳定性和可靠性。

目前支持的 Tier 1 翻译引擎包括：
1. SiliconFlowFree
2. OpenAI
3. AliyunDashScope
4. DeepSeek
5. SiliconFlow
6. Zhipu
7. OpenAICompatible

#### Tier 2（社区支持）

**Tier 2 翻译引擎**由社区维护和支持。
当这些引擎遇到问题时，项目维护者不会直接提供修复。相反，他们会将相关问题标记为 `help wanted`，并欢迎贡献者提交拉取请求来帮助解决。

所有程序支持但未明确列在 Tier 1 下的引擎均被视为 Tier 2 翻译引擎。

#### 已弃用的引擎

以下翻译引擎已被**弃用**，将不再维护或支持：

1. Bing
2. Google
