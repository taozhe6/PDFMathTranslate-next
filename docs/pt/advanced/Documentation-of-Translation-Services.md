### Visualizando Serviços de Tradução Disponíveis via Linha de Comando

Você pode confirmar os serviços de tradução disponíveis e seu uso imprimindo a mensagem de ajuda na linha de comando.

```bash
pdf2zh_next -h
```

No final da mensagem de ajuda, você pode visualizar informações detalhadas sobre os diferentes serviços de tradução.

### Política de Suporte para Mecanismos de Tradução

#### Tier 1 (Suporte oficial)

**Motores de tradução Tier 1** são mantidos diretamente pelos mantenedores do projeto. Embora os mantenedores **não usem este projeto regularmente**, eles confiarão em problemas do GitHub para identificar problemas. Quando qualquer um desses mecanismos encontrar problemas, os mantenedores os corrigirão o mais rápido possível para garantir estabilidade e confiabilidade.

Atualmente, os motores de tradução Tier 1 suportados incluem:
1. SiliconFlowFree
2. OpenAI
3. AliyunDashScope
4. DeepSeek
5. SiliconFlow
6. Zhipu
7. OpenAICompatible

#### Tier 2 (Suporte da Comunidade)

**Tier 2 translation engines** são mantidos e suportados pela comunidade.  
Quando esses mecanismos encontram problemas, os mantenedores do projeto não fornecerão correções diretas. Em vez disso, eles rotularão os problemas relacionados com `help wanted` e darão as boas-vindas a pull requests de contribuidores para ajudar a resolvê-los.

Todos os mecanismos que são suportados pelo programa, mas não listados explicitamente sob Tier 1, são considerados Tier 2 translation engines.

#### Motores Descontinuados

Os seguintes mecanismos de tradução foram **descontinuados** e não serão mais mantidos ou suportados:

1. Bing
2. Google
