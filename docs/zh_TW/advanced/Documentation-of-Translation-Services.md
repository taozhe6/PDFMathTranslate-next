### 通過命令行查看可用的翻譯服務

您可以通過在命令行中打印幫助訊息來確認可用的翻譯服務及其使用方法。

```bash
pdf2zh_next -h
```

在幫助訊息的最後，您可以查看不同翻譯服務的詳細資訊。

### 翻譯引擎支持政策

#### Tier 1 (官方支持)

**Tier 1 翻譯引擎**由項目維護者直接維護。雖然維護者**不經常使用此項目**，但他們會依賴 GitHub issues 來發現問題。當這些引擎中的任何一個遇到問題時，維護者將盡快修復，以確保穩定性和可靠性。

目前支持的 Tier 1 翻譯引擎包括：
1. SiliconFlowFree
2. OpenAI
3. AliyunDashScope
4. DeepSeek
5. SiliconFlow
6. Zhipu
7. OpenAICompatible

#### Tier 2（社區支持）

**Tier 2 翻譯引擎**由社區維護和支持。  
當這些引擎遇到問題時，項目維護者不會直接提供修復。相反，他們會將相關問題標記為 `help wanted`，並歡迎貢獻者提交 pull request 來幫助解決。

所有由程序支持但未明確列在 Tier 1 下的引擎都被視為第二級翻譯引擎。

#### 已棄用的引擎

以下翻譯引擎已被**棄用**，將不再進行維護或支持：

1. Bing
2. Google
